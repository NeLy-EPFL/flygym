## Additional guided tutorials

The [NeuroMechFly v2 paper](https://www.nature.com/articles/s41592-024-02497-y.epdf) was accompanied by 13 detailed tutorials. These tutorials rely on the `flygym.guided` API, which offers a more user-friendly and guided interface at the cost of computational efficiency. You can use these tutorials to learn about NeuroMechFly and neuromechanical simulation in general. However, for scientific studies, we recommend using the newer `flygym` or `flygym.warp` APIs. See [the difference here](https://neuromechfly.org/#choose-your-own-flavor).
