# Tutorials

If you are new to FlyGym, we recommend that you get started with tutorials under the "Tutorials" section in the navigation bar.

## Additional guided tutorials

The [NeuroMechFly v2 paper](https://www.nature.com/articles/s41592-024-02497-y.epdf) was accompanied by 13 detailed tutorials. These tutorials rely on the legacy [`flygym-gymnasium` API](https://github.com/NeLy-EPFL/flygym-gymnasium), which offers a more user-friendly and guided interface at the cost of computational efficiency. You can use these tutorials to learn about NeuroMechFly and neuromechanical simulation in general. However, for scientific studies, we recommend using the newer API documented on this site. See [the difference here](https://neuromechfly.org/migration).
